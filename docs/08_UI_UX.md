# UI / UX Design

## Design Goals

The application should be:

* Modern
* Responsive
* Clean
* Fast
* Simple
* Easy for non-technical users

---

# Theme

Use Tailwind CSS.

Design Philosophy

- Modern
- Minimal
- Professional
- Clean
- Responsive
- Accessible

Use utility-first styling.

Avoid inline styles.

Create reusable UI components.

Maintain a consistent design system across the application.

Primary Color

Blue

Secondary

White

Accent

Green

Status Colors

Success

Green

Warning

Orange

Error

Red

Gray
