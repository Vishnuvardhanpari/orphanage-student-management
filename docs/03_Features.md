# Student Management

This is the primary module of the application.

## Student Registration

- Add Student
- Upload Student Photo
- Upload Supporting Documents
- Save Student Record

## Student Profile

- View Complete Student Information
- View Student Photo
- View Uploaded Documents
- Download Documents

## Student Update

- Edit Student Information
- Replace Student Photo
- Upload Additional Documents
- Replace Existing Documents

## Student Exit

- Soft Delete Student
- Preserve Historical Record
- Record Exit Details

## Student Restore

Administrator can restore soft deleted students.

---

# Student Documents

Documents are managed within the Student Profile.

Users can:

- Upload Documents during Student Registration
- Upload Additional Documents during Student Edit
- Preview Documents
- Download Documents
- Replace Existing Documents
- Remove Documents if uploaded incorrectly

Supported File Types

- PDF
- JPG
- JPEG
- PNG
